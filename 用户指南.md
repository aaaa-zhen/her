# 你只需要做 2 件事

---

## Mac 用户

### 第一步：开启远程登录

打开 **系统设置 → 通用 → 共享 → 远程登录 → 打开**

### 第二步：运行配置脚本

打开 **终端**（在 Launchpad 搜索"终端"），粘贴下面这行，按回车：

```
bash ~/Desktop/mac-setup.sh
```

完成后按脚本提示打开浏览器即可。

---

## Windows 用户

### 第一步：运行配置脚本

找到桌面上的 **windows-setup.ps1** 文件，**右键 → 用 PowerShell 以管理员身份运行**

脚本会自动完成所有配置（安装 SSH、建文件夹、下载工具、连接服务器）。

> 如果弹出"此应用无法运行"，在 PowerShell 里运行：
> ```
> Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
> ```
> 然后重新右键运行脚本。

---

## 以后每次重启电脑

重新运行一次对应的脚本即可（Mac：终端运行命令；Windows：右键运行 .ps1）。
